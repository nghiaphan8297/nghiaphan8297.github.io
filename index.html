Has design become too hard?

Digital design has changed because of the advancedchange of technology that make designers have to master more and more tools. Some designers may say that the “fun” has gone out of it. An example of this is the obstacles of web design. In the early, pixels were not standardized between PC and Macintosh; monitor aspect ratios were not standardized. And when we got a standardized layout language for the web, browser supported it differently for years. Then, preference settings, outdated browser, wonky operating system,…

Design was never easy, and it’s getting harder. Designers say that today’s code is too hard. But was it ever easy? Today, we’re using many forms of device so, instead of pretending or wishing that everyone used the same platform with the same setting, designers have to design accordingly. Then browsers support web standard well enough for us to use them. And thank to that, once we had mastered our CSS, HTML and JavaScript, we wouldn’t need to learn much anything new. Surely, there are other frameworks that could be better but you don’t need to learn much more than that. You dont have to know every framework.
